# SEPN_Electricity_Tariff_Data_Analytics

This project analyses electricity network tariff data to understand how charges vary across customers and over time.


## Business Problem

Energy tariff structures can be complex, making it difficult to understand:

- How charges differ across customer groups  
- How pricing changes over time  
- Which segments contribute most to costs  


## Approach

I used SQL to analyse tariff data across multiple years, focusing on customer classification and cost patterns.

The results were visualised in Power BI for clarity.


## Key Insights

- Tariff charges vary significantly by customer demand level  
- Certain customer groups contribute more to overall costs  
- Year-on-year changes highlight pricing trends
